About Me

A Full Stack Developer and Computer Science Engineering Grad from Pune, India.

I am Ketan Arwat, working at Cognizant Technology Solutions as a growing Full Stack Developer. I follow detail-oriented approach, where tiny details in design and code matter a lot and go a long way.

Projects I have built

PubG Tour

A portal where users can browse through PubG tournaments and the matches from the Official PubG API. A project built using Spring Boot, Angular 7, Angular Material, MySQL and Docker

Movie Cruizer

Users can create an account and explore popular and top-rated movie from the TMDB API and add movies to their watchlist. A project built using Spring Boot, Angular 7, Bootstrap 4, MySQL and Docker.

DroolZone

A basic online food ordering portal where registered users can 'drool' through various food items and add them to their online carts. A project built using Spring MVC, Angular 7, Bootstrap and MySQL.

File Share

A light-weight portal with registered users can send and recieve files from eachother. A project built using HTML 5, CSS, Servlet, JSP and MySQL.
